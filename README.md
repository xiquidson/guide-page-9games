# guide-page-9games

## 项目简介

`guide-page-9games` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，旨在以静态页面的形式整理和存放一些辅助性、说明性的内容。每个页面均为独立的 HTML 文件，可直接通过浏览器查看。

## 目录说明

```
guide-page-9games/
├── README.md          # 本文件
├── pages/             # 存放所有 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/            # 页面所需的静态资源（如 CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立的 HTML 页面存放于此，命名清晰，便于查找。
- `assets/`：页面依赖的样式、脚本和图片等资源，按类型分目录管理。

## 页面归档说明

本仓库中的页面均为独立的 HTML 文件，可直接打开运行。每个页面都包含完整的 HTML 结构，不依赖外部数据库或后端服务。页面内容可能涉及指引、说明或辅助信息，适合作为静态资源存档使用。

- 页面之间无强关联，可单独使用。
- 所有页面均采用标准 HTML5 编写，兼容主流现代浏览器。
- 如有资源依赖，请确保 `assets/` 目录下的文件路径正确。
